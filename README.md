# DartVista

Im Programm sind zwei Dartpfeilmodelle enthalten, welche mittels Simulation auf ein Board geschossen werden können. Die Animation kann vom Nutzer beliebig angepasst werden. Ausserdem kann durch Scrollen mit dem Mausrad gezoomt werden.

## Installation

Windows 11:

Für die Nutzung von DartVista muss [Python 3.12 oder neuer](https://www.python.org/downloads/) heruntergeladen und installiert werden.
Am besten überprüfen Sie, ob Python richtig installiert wurde:
```bash
python --version
```

Nachdem Python installiert wurde, muss Ursina installiert werden.

```bash
python -m pip install ursina
```

Nun kann DartVista durch Ausführen der dartvista.exe gestartet werden
