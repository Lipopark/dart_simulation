# Dart Simulation

Dieses Programm beinhaltet zwei Dartpfeilmodelle, welche mittels Simulation auf ein Board geschossen werden können. Die Animation kann vom Nutzer beliebig angepasst werden.

## Nutzung

Windows 11:

Für die Nutzung des Programms muss [Python 3.12 oder neuer](https://www.python.org/downloads/) heruntergeladen werden.

Dann kann das Programm durch Ausführen der main.exe gestartet werden.
