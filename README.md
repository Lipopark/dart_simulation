# Dart Simulation

Dieses Programm beinhaltet zwei Dartpfeilmodelle, welche mittels Simulation auf ein Board geschossen werden können. Die Animation kann vom Nutzer beliebig angepasst werden.

## Nutzung

Windows 11: 
Die .zip-Datei von Git Hub muss entpackt werden.
Um das Programm zu starten müssen Python und ursina engine installiert sein.

[Python Releases](https://www.python.org/downloads/windows/)

```bash
python -m pip install ursina
```

Sind Python und ursina engine installiert, kann das Programm durch Ausführung der main.py gestartet werden. Dafür im Hauptverzeichnis ein Terminal öffnen und folgenden Befehl einfügen:

```python
python main.py
```
